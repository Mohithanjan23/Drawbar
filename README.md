# DrawBar

DrawBar is a modern, web-based design platform inspired by Canva. It allows users to create, edit, and share graphic designs using an intuitive interactive canvas.

##  Features

- **Interactive Canvas**: Drag, drop, and resize shapes, text, and images.
- **Supabase Integration**: Secure authentication and cloud storage for your projects.
- **Tailwind CSS**: A sleek, modern user interface.
- **AI-Powered (Planned)**: Future integration for AI-generated layouts and image editing.

##  Tech Stack

- **Frontend**: Next.js (React)
- **Styling**: Tailwind CSS
- **Backend/Database**: Supabase
- **Language**: TypeScript
